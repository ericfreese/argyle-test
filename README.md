# Argyle Sandbox

Play with [Argyle](https://github.com/ericfreese/argyle).

## Running locally

Requires ruby, npm, and bower.

```
git clone https://github.com/ericfreese/argyle-test.git
cd argyle-test
bundle install
npm install
bower install
gulp dev
```

Gulp should serve `argyle-test/dist/` at `http://localhost:8080`.
